# 최대공약수 구하기
```{.python}
def GCD(a, b):
    r = a % b
    if r == 0:
        return b
    else:
        return GCD(b, r)
```
### 종료 조건
a' % b'의 나머지가 0일 때.  
a'가 b'로 나누어 떨어졌을 때,  
역재귀적으로 이전 단계의 a'위치에 b'의 배수를 넣으면 거슬러 올라가면  
최초로 나누어지는 수 a는 b'로 나누어 떨어짐을 알 수 있다.  
최초로 나누는 수 b도 b'로 나누어 떨어지며, 따라서 최대공약수다.  

### a와 b의 대소
만약 a < b이면,  
r = a % b = a가 되고,  
GCD(b, r) = GCD(b, a)가 되어 자동으로 알맞게 진행된다.
